## 注意事项

1. **Python 版本**: 开发环境使用 Python 3.14，Dockerfile 固定为 3.11-slim（与设计文档一致）
2. **Docker CMD**: 容器内使用 `src.main:app` 而非 `backend.src.main:app`，因为 WORKDIR 即为 backend 内容
3. **路由为占位**: 当前路由返回固定 JSON，实际代理逻辑将在后续任务中实现
4. **无 .env 文件**: pydantic-settings 虽已引入但尚未使用，后续任务中配置管理将使用
