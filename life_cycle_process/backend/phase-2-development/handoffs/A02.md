## config.py 设计要点

- **BaseSettings 子类**：字段名 snake_case 自动映射 UPPER_CASE 环境变量
- **model_config**：指定 `env_file = ".env"`，支持 .env 文件加载
- **validate_required()**：检查 `VISION_API_KEY` 和 `TARGET_DEFAULT_API_KEY` 非空，缺失时抛出 ValueError
- **默认值对齐 Coding Plan**：
  - vision_base_url: `https://coding.dashscope.aliyuncs.com`
  - vision_model: `qwen3.7-plus`
  - target_default_base_url: `https://ark.cn-beijing.volces.com/api/coding`

## 注意事项

1. **validate_required() 为显式调用**：ProxyConfig 构造时不自动校验，由调用方（如 app startup）显式调用 validate_required()
2. **env file 位置**：pydantic-settings 从当前工作目录查找 .env 文件，运行时需确保 cwd 正确
3. **TargetModelConfig 为纯 dataclass**：不继承 BaseSettings，从 ProxyConfig 字段构造，测试中手动从 os.environ 取值验证
4. **Coding Plan URL 对齐**：vision_base_url 和 target_default_base_url 均使用 Coding Plan 专属地址（非通用 API 地址），与 api-integration-reference.md 一致
5. **GLM 配置预留**：target_glm_* 字段已定义但本任务未测试，后续任务使用
