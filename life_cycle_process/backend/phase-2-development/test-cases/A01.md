| 验证方式 | 要求等级 | 工具/命令 | 说明 |
|----------|---------|-----------|------|
| 编译检查 | ✅ 必需 | `python -m py_compile` | 零语法错误 |

## TC-A01-BLD-001: 编译 — 入口文件语法正确

- **Given**: `backend/src/main.py` 存在
- **When**: `python -m py_compile backend/src/main.py`
- **Then**: 编译成功，exit code 0

## TC-A01-BLD-002: 编译 — FastAPI app 创建无错误

- **Given**: `backend/src/app.py` 存在
- **When**: `python -c "from backend.src.app import app; assert app.title != ''"`
- **Then**: app 实例创建成功，title 非空

## TC-A01-BLD-003: 依赖 — requirements.txt 可安装

- **Given**: `backend/requirements.txt` 存在
- **When**: `pip install -r backend/requirements.txt --dry-run`
- **Then**: 所有依赖可解析，无冲突

## TC-A01-BLD-004: Docker — Dockerfile 语法正确

- **Given**: `backend/Dockerfile` 存在
- **When**: `docker build --check .` 或检查 FROM/COPY/CMD 指令
- **Then**: Dockerfile 指令合法

## TC-A01-BLD-005: 项目结构 — 所有预期目录存在

- **Given**: 项目已初始化
- **When**: `ls backend/src/ backend/tests/`
- **Then**: `src/` 和 `tests/` 目录均存在

## TC-A01-LOG-001: 逻辑 — FastAPI app 包含所有路由

- **Given**: app 实例已创建
- **When**: 检查 `app.routes`
- **Then**: 路由列表包含 `/v1/messages`、`/v1/chat/completions`、`/health`

## TC-A01-API-001: API — 服务启动后 GET / -> 200

- **Given**: 服务已启动 (`uvicorn backend.src.main:app`)
- **When**: `curl http://localhost:8080/`
- **Then**: 返回 200 或 404（非 500）
