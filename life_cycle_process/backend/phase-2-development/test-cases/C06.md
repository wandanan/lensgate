## TC-C06-API-001: API — 缺失 x-api-key → 401

- **Given**: 服务配置 PROXY_API_KEY="secret"
- **When**: POST /v1/messages 不带 x-api-key
- **Then**: 返回 401
- **PASS 判定**: status 401
- **FAIL 判定**: 200 或 500

## TC-C06-API-002: API — 无效 x-api-key → 401

- **Given**: 服务配置 PROXY_API_KEY="secret"
- **When**: POST /v1/messages，x-api-key: wrong
- **Then**: 返回 401
- **PASS 判定**: status 401
- **FAIL 判定**: 200

## TC-C06-API-003: API — 有效 x-api-key → 继续处理

- **Given**: 服务配置 PROXY_API_KEY="secret"
- **When**: POST /v1/messages，x-api-key: secret
- **Then**: 返回非 401（正常处理）
- **PASS 判定**: status != 401
- **FAIL 判定**: 401

## TC-C06-API-004: API — Content-Length > 10MB → 413

- **Given**: 请求 body 12MB
- **When**: POST /v1/messages
- **Then**: 返回 413
- **PASS 判定**: status 413
- **FAIL 判定**: 正常处理或 500
