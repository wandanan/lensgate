## TC-C01-BLD-001: 编译检查

- **Given**: `backend/src/model_router.py` 存在
- **When**: `python -m py_compile backend/src/model_router.py`
- **Then**: 编译成功

## TC-C01-LOG-001: 逻辑 — 根据 model 名匹配配置

- **Given**: router 配置 "deepseek" → deepseek config, "glm" → glm config
- **When**: `router.resolve("deepseek-chat")`
- **Then**: 返回 deepseek config (model_id="deepseek-chat")

## TC-C01-LOG-002: 逻辑 — 不匹配时 fallback 到默认

- **Given**: router 默认配置指向 deepseek
- **When**: `router.resolve("unknown-model")`
- **Then**: 返回默认配置

## TC-C01-LOG-003: 逻辑 — 大小写不敏感匹配

- **Given**: router 配置含 "deepseek"
- **When**: `router.resolve("DeepSeek-Chat")`
- **Then**: 返回 deepseek 配置

## TC-C01-LOG-004: 逻辑 — TargetModelConfig 字段完整性

- **Given**: 创建 TargetModelConfig
- **When**: 检查字段
- **Then**: 含 model_id, api_base, api_key, extra_params

## TC-C01-API-001: API — GET /health 返回 200

- **Given**: 服务运行中
- **When**: `curl http://localhost:8080/health`
- **Then**: 返回 200，body 含 `"status": "ok"`

## TC-C01-API-002: API — /health 含版本信息

- **Given**: 服务运行中
- **When**: GET /health
- **Then**: body 含 `"version"` 和 `"timestamp"` 字段
