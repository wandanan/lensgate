## TC-B02-LOG-001: 逻辑 — has_images 对纯文本请求返回 False

- **Given**: ProxyRequest，messages 中只有 TextBlock
- **When**: 调用 `has_images(request)`
- **Then**: 返回 False

## TC-B02-LOG-002: 逻辑 — has_images 对含图请求返回 True

- **Given**: ProxyRequest，messages[0].content 含 ImageBlock
- **When**: 调用 `has_images(request)`
- **Then**: 返回 True

## TC-B02-LOG-003: 逻辑 — extract_images 返回所有 ImageBlock

- **Given**: ProxyRequest，messages 中 3 个 ImageBlock（分布在2条消息中）
- **When**: 调用 `extract_images(request)`
- **Then**: 返回 list，长度 3，每个元素为 ImageBlock

## TC-B02-LOG-004: 逻辑 — Anthropic base64 source 正确解码

- **Given**: Anthropic image block，`source.type="base64"`, `source.data="aGVsbG8="`（hello）
- **When**: 调用 extract_images
- **Then**: ImageBlock.image_data == b"hello", source_type="base64"

## TC-B02-LOG-005: 逻辑 — ImageBlock 含 message_index 和 block_index

- **Given**: ProxyRequest，图片在第 1 条消息的第 2 个 content block
- **When**: extract_images
- **Then**: ImageBlock.message_index == 1, block_index == 2

## TC-B02-LOG-006: 逻辑 — media_type 从 Anthropic source 提取

- **Given**: Anthropic image block，`source.media_type="image/jpeg"`
- **When**: 提取
- **Then**: ImageBlock.media_type == "image/jpeg"

## TC-B02-LOG-007: 逻辑 — OpenAI data URI 解析

- **Given**: OpenAI image_url，`url="data:image/png;base64,aGVsbG8="`
- **When**: 提取
- **Then**: ImageBlock.image_data == b"hello", media_type="image/png", source_type="data_uri"

## TC-B02-LOG-008: 逻辑 — extract_images 返回空列表（无图）

- **Given**: 纯文本 ProxyRequest
- **When**: extract_images
- **Then**: 返回空列表 `[]`

## TC-B02-SYS-001: 系统行为 — 有图片进入 EXTRACTING

**来源：系统交互规格 §4 — IMAGE_CHECK → EXTRACTING**

- **Given**：ProxyRequest 含 image block
- **When**：管道执行 IMAGE_CHECK
- **Then**：has_images 返回 True，extract_images 返回非空列表
- **PASS 判定**：has_images() == True，len(extract_images()) >= 1
- **FAIL 判定**：has_images() == False

## TC-B02-SYS-002: 系统行为 — 无图片跳转到 FORWARDING

**来源：系统交互规格 §4 — IMAGE_CHECK → FORWARDING**

- **Given**：ProxyRequest 不含 image block
- **When**：管道执行 IMAGE_CHECK
- **Then**：has_images 返回 False，跳过后续阶段
- **PASS 判定**：has_images() == False
- **FAIL 判定**：误判为有图片

## TC-B02-SYS-003: 系统行为 — URL 图片下载失败注入降级文本

**来源：系统交互规格 §4 — EXTRACTING → FORWARDING (降级)**

- **Given**：ImageBlock 的 source 为 URL，下载超时
- **When**：extract_images 下载
- **Then**：该图片标记为下载失败
- **PASS 判定**：image_data 为 None，标记 is_error=True
- **FAIL 判定**：抛出未捕获异常

## TC-B02-SYS-004: 系统行为 — 合法图片格式通过

**来源：系统交互规格 §6 — 图片格式白名单（正常路径）**

- **Given**：ImageBlock media_type="image/png"
- **When**：validate_image_format(block)
- **Then**：返回 True
- **PASS 判定**：validate_image_format 返回 True
- **FAIL 判定**：返回 False

## TC-B02-SYS-005: 系统行为 — 非法图片格式 → 400

**来源：系统交互规格 §6 — 图片格式白名单（违反路径）**

- **Given**：ImageBlock media_type="image/bmp"（不支持）
- **When**：validate_image_format(block) 或管道中校验
- **Then**：返回 400，body 含 "unsupported_media_type"
- **PASS 判定**：HTTP status 400
- **FAIL 判定**：允许通过或 500
