## TC-C07-API-001: API — 无效 JSON body → 400

- **Given**: 服务运行中
- **When**: POST /v1/messages，body="not valid json"
- **Then**: 400，body 含 "invalid_request"
- **PASS 判定**: status 400 + "invalid_request" in body
- **FAIL 判定**: 500

## TC-C07-API-002: API — 目标模型 500 → 503

- **Given**: mock Target API 500
- **When**: 纯文本请求
- **Then**: 503，body 含 "target_model_unavailable"
- **PASS 判定**: status 503 + error 消息
- **FAIL 判定**: 500 透传

## TC-C07-API-003: API — Vision API 失败不阻断请求

- **Given**: mock Vision API 500，mock Target API 200
- **When**: 含图请求
- **Then**: 200，target body 含降级文本 "[图片无法识别]"
- **PASS 判定**: status 200 + body 含降级文本
- **FAIL 判定**: 500

## TC-C07-API-004: API — 目标模型超时 → 504

- **Given**: mock Target API 触发 TimeoutException
- **When**: 请求
- **Then**: 504，body 含 "target_model_timeout"
- **PASS 判定**: status 504
- **FAIL 判定**: 其他错误码

## TC-C07-SYS-001: 系统行为 — §7.1: 客户端断连取消上游

**来源：系统交互规格 §7.1**

- **Given**：流式转发中，客户端断开
- **When**：检测到 disconnect
- **Then**：上游 httpx stream 关闭
- **PASS 判定**：无连接泄漏，无未捕获异常
- **FAIL 判定**：连接残留或 crashed

## TC-C07-SYS-002: 系统行为 — §7.1: Qwen 429 重试

**来源：系统交互规格 §7.1 — Qwen API 限流**

- **Given**：mock Qwen 连续 429
- **When**：识图
- **Then**：重试 1 次后降级，最终 200
- **PASS 判定**：API 调用次数=2，最终 status 200
- **FAIL 判定**：未重试或返回 429

## TC-C07-SYS-003: 系统行为 — §7.1: 配置缺失启动检查
