## TC-C02-BLD-001: 编译检查

- **Given**: `backend/src/vision_client.py` 存在
- **When**: `python -m py_compile backend/src/vision_client.py`
- **Then**: 编译成功

## TC-C02-LOG-001: 逻辑 — recognize 发送正确的 API 请求

- **Given**: mock httpx 返回 200 + `{"choices":[{"message":{"content":"描述文字"}}]}`
- **When**: `vision_client.recognize(image_block)`
- **Then**: 发送到 `https://coding.dashscope.aliyuncs.com/v1/chat/completions`，model="qwen3.7-plus"，content 含 image_url

## TC-C02-LOG-002: 逻辑 — recognize 返回非空字符串

- **Given**: mock Vision API 返回 200
- **When**: `recognize(image_block)`
- **Then**: 返回非空 str

## TC-C02-LOG-003: 逻辑 — image_data 正确 base64 编码传输

- **Given**: ImageBlock image_data=b"test-image-bytes", media_type="image/png"
- **When**: recognize
- **Then**: 请求 body 中 image_url.url = "data:image/png;base64,dGVzdC1pbWFnZS1ieXRlcw=="

## TC-C02-LOG-004: 逻辑 — recognize_batch 并行调用

- **Given**: 3 个 ImageBlock，mock Vision API 每次耗时 100ms
- **When**: `recognize_batch([img1, img2, img3])`
- **Then**: 总耗时 < 150ms（并行），返回 3 个结果

## TC-C02-LOG-005: 逻辑 — recognize_batch 单个失败不阻塞其他

- **Given**: 3 个 ImageBlock，第 2 个的 API 返回 500
- **When**: recognize_batch
- **Then**: 返回 3 个结果，其中 [1] = "[图片无法识别]"，[0] 和 [2] 正常

## TC-C02-LOG-006: 逻辑 — Vision API 超时返回降级文本

- **Given**: mock Vision API 延迟 35s（超时设为 30s）
- **When**: recognize(img)
- **Then**: 返回 "[图片无法识别]"

## TC-C02-SYS-001: 系统行为 — §4: RECOGNIZING API 成功

**来源：系统交互规格 §4 — RECOGNIZING → REWRITING**

- **Given**：mock Vision API 返回 200
- **When**：recognize 调用
- **Then**：返回文字描述，长度 > 0
- **PASS 判定**：结果非空字符串
- **FAIL 判定**：返回空或 None

## TC-C02-SYS-002: 系统行为 — §4: RECOGNIZING API 失败降级

**来源：系统交互规格 §4 — RECOGNIZING → FORWARDING (降级)**

- **Given**：mock Vision API 返回非 200
- **When**：recognize 调用
- **Then**：返回 "[图片无法识别]"
- **PASS 判定**：返回降级文本，不抛异常
- **FAIL 判定**：抛异常或返回 None

## TC-C02-SYS-003: 系统行为 — API 429 重试后成功

- **Given**：第 1 次 429，延迟 1s，第 2 次 200
- **When**：recognize
- **Then**：第 2 次成功，返回正常文字
- **PASS 判定**：结果非空，总共调用了 2 次 API
- **FAIL 判定**：第 1 次 429 直接放弃

## TC-C02-SYS-004: 系统行为 — API 429 重试仍失败降级

- **Given**：两次调用均返回 429
- **When**：recognize
- **Then**：返回 "[图片无法识别]"，总共调用了 2 次 API
- **PASS 判定**：降级文本 + 2 次 API 调用
- **FAIL 判定**：无限重试或抛异常
