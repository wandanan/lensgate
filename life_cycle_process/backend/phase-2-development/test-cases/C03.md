## TC-C03-BLD-001: 编译检查

- **Given**: `backend/src/request_rewriter.py` 存在
- **When**: `python -m py_compile backend/src/request_rewriter.py`
- **Then**: 编译成功

## TC-C03-LOG-001: 逻辑 — 单个 image block 被替换为 text block

- **Given**: Anthropic messages[0].content = [ImageBlock(...), TextBlock("这是啥")]
- **When**: `rewriter.rewrite(request, [(img, "一张包含文字的截图")])`
- **Then**: 输出 messages[0].content 不含 ImageBlock，只有 text block
- **PASS 判定**: 所有 ImageBlock 均被替换，text block 含 "[图片 1/1 的描述：一张包含文字的截图]"

## TC-C03-LOG-002: 逻辑 — 多个 image block 替换为带编号的 text block

- **Given**: 3 个 ImageBlock，识图结果 ["结果A", "结果B", "结果C"]
- **When**: rewrite
- **Then**: text block 含 "[图片 1/3 的描述：结果A]", "[图片 2/3 的描述：结果B]", "[图片 3/3 的描述：结果C]"

## TC-C03-LOG-003: 逻辑 — 非图片 content 保持不变

- **Given**: messages 含 TextBlock("hello") + ImageBlock + TextBlock("world")
- **When**: rewrite（只替换 ImageBlock）
- **Then**: TextBlock("hello") 和 TextBlock("world") 依然存在且内容不变

## TC-C03-LOG-004: 逻辑 — 重写保持 source_format

- **Given**: Anthropic 格式请求
- **When**: rewrite
- **Then**: 输出 body 仍为 Anthropic 格式（messages, model, max_tokens 结构不变）

## TC-C03-LOG-005: 逻辑 — 只有图片无文字的 content → 仅转发识图结果

- **Given**: messages[0].content = [ImageBlock]
- **When**: rewrite
- **Then**: messages[0].content = [TextBlock("[图片 1/1 的描述：...]")]
- **PASS 判定**: content 含 1 个 TextBlock，内容为识图结果

## TC-C03-SYS-001: 系统行为 — §4: REWRITING 正确替换

**来源：系统交互规格 §4 — REWRITING**

- **Given**：ProxyRequest 含 2 个 ImageBlock，vision_results 对应 2 个描述
- **When**：rewrite
- **Then**：输出 messages 中 0 个 ImageBlock，2 个 TextBlock 含描述
- **PASS 判定**：`count(ImageBlock) == 0` and `count("图片") == 2`
- **FAIL 判定**：仍残留 ImageBlock

## TC-C03-SYS-002: 系统行为 — §7.2: 仅图片无文字 content

**来源：系统交互规格 §7.2 — 空状态 — 无文本仅图片**

- **Given**：messages[0].content = [ImageBlock]（无 TextBlock）
- **When**：rewrite 完成
- **Then**：目标模型收到纯文本，内容为识图描述
- **PASS 判定**：rewritten messages[0].content 只有 text block，无 image block
- **FAIL 判定**：rewritten body 为空或无 text
