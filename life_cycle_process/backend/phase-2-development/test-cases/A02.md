## TC-A02-BLD-001: 编译 — config.py 语法正确

- **Given**: `backend/src/config.py` 存在
- **When**: `python -m py_compile backend/src/config.py`
- **Then**: 编译成功

## TC-A02-BLD-002: 编译 — models.py 语法正确

- **Given**: `backend/src/models.py` 存在
- **When**: `python -m py_compile backend/src/models.py`
- **Then**: 编译成功

## TC-A02-LOG-001: 逻辑 — ProxyConfig 从环境变量加载

- **Given**: 设置 `PROXY_PORT=9090`
- **When**: 创建 `ProxyConfig()` 实例
- **Then**: `config.proxy_port == 9090`

## TC-A02-LOG-002: 逻辑 — ProxyConfig 默认值

- **Given**: 未设置任何环境变量
- **When**: 创建 `ProxyConfig()` 实例
- **Then**: `proxy_host == "0.0.0.0"`, `proxy_port == 8080`, `vision_timeout == 30`

## TC-A02-LOG-003: 逻辑 — ProxyRequest 创建与字段访问

- **Given**: import ProxyRequest, Message, TextBlock
- **When**: 构造 `ProxyRequest(source_format="anthropic", target_model="deepseek", messages=[Message(role="user", content=[TextBlock(text="hello")])])`
- **Then**: `request.source_format == "anthropic"`, `len(request.messages) == 1`

## TC-A02-LOG-004: 逻辑 — Message role 限定值

- **Given**: Message 类定义 role 为 Literal["user", "assistant", "system"]
- **When**: 创建 `Message(role="user", content=[TextBlock(text="hi")])`
- **Then**: 创建成功，无异常

## TC-A02-LOG-005: 逻辑 — 目标模型配置加载

- **Given**: 设置 `TARGET_DEFAULT_MODEL=deepseek`, `TARGET_DEFAULT_BASE_URL=https://test.com`, `TARGET_DEFAULT_API_KEY=key123`
- **When**: 创建 TargetModelConfig 从环境变量
- **Then**: `model_id == "deepseek"`, `api_base == "https://test.com"`, `api_key == "key123"`

## TC-A02-LOG-006: 逻辑 — TextBlock 和 ImageBlock 为 ContentBlock union

- **Given**: ContentBlock = Union[TextBlock, ImageBlock]
- **When**: 创建 `TextBlock(text="hello")` 和 `ImageBlock(image_data=b"abc", media_type="image/png", ...)`
- **Then**: 两者均可作为 ContentBlock 使用

## TC-A02-LOG-007: 逻辑 — 配置缺失时抛异常

- **Given**: `VISION_API_KEY` 和 `TARGET_DEFAULT_API_KEY` 均未设置
- **When**: 调用 `config.validate_required()`
- **Then**: 抛出 RuntimeError 或 ValueError
