- **When**: 调用 `detect_format(path, body)`
- **Then**: 返回 `"anthropic"`

- **Given**: 请求 path 为 `/v1/chat/completions`
- **When**: 调用 `detect_format(path, body)`
- **Then**: 返回 `"openai"`

## TC-B01-LOG-003: 逻辑 — 解析 Anthropic 纯文本请求

- **Given**: Anthropic body: `{"model":"claude","max_tokens":100,"messages":[{"role":"user","content":"hello"}]}`
- **When**: 调用 `parse_anthropic_request(body)`
- **Then**: 返回 ProxyRequest，`source_format="anthropic"`，`messages[0].content` 为 1 个 TextBlock

## TC-B01-LOG-004: 逻辑 — 解析 Anthropic 多 content block 请求

- **When**: 调用 `parse_anthropic_request(body)`
